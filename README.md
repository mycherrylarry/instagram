実装した機能
====
*他のユーザをフォロー/解除できる
*写真がアップロードできる
*フォローしたユーザの写真がフォトストリームで閲覧できリアルタイムで更新される-- Notification 
*写真がクロップされる
*外部サービス（FACEBOOK）と連携して投稿できる 
*画像加工が出来る 
*いいね(like)が出来る
*ユーザーコメントが出来る
*ユーザーの情報管理と写真管理
